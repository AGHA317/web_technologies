50 Questions and Answers

  1. What is HTML? - HTML stands for HyperText Markup Language.
  2. What is the purpose of CSS? - CSS is used for styling web pages.
  3. What is JavaScript? - JavaScript is a scripting language for web development.
  4. What is an HTML tag? - An HTML tag defines elements in a document.
  5. What is a CSS selector? - A CSS selector targets HTML elements for styling.
  6. What does the <title> tag do? - It defines the title of the webpage.
  7. What is the difference between ID and Class in CSS? - ID is unique, while Class can be used multiple times.
  8. What is a div in HTML? - A div is a container for grouping elements.
  9. What is the function of the <head> section? - It contains metadata and links to styles and scripts.
  10. What is a hyperlink in HTML? - A hyperlink is created using the <a> tag.
  11. What does the <p> tag represent? - It defines a paragraph.
  12. What is an HTML attribute? - Attributes provide additional information about an element.
  13. What is the purpose of the <meta> tag? - It provides metadata about an HTML document.
  14. What does the <img> tag do? - It embeds images in an HTML document.
  15. What is the purpose of the <table> tag? - It defines a table structure.
  16. How do you create a list in HTML? - Using <ul> for unordered lists and <ol> for ordered lists.
  17. What is responsive design? - Designing web pages to work on various devices.
  18. What is a favicon? - A small icon displayed in the browser tab.
  19. What is an iframe? - An iframe embeds another document inside an HTML page.
  20. What is Bootstrap? - A CSS framework for responsive design.
  21. What is an event in JavaScript? - An event is an action that triggers code execution.
  22. What is the difference between inline, internal, and external CSS? - Inline applies directly, internal is within <style>, and external is in a separate file.
  23. What is the purpose of JavaScript functions? - To execute reusable blocks of code.
  24. What is a semantic tag in HTML? - Tags like <header>, <footer>, and <article> that describe content meaningfully.
  25. What is the difference between GET and POST methods? - GET sends data in the URL, while POST sends data securely in the body.
  26. How do you center text using CSS? - Using `text-align: center;`.
  27. What is the box model in CSS? - It includes margin, border, padding, and content.
  28. What is the difference between absolute and relative positioning in CSS? - Absolute positions relative to the nearest positioned ancestor, relative adjusts from its normal position.
  29. What is flexbox? - A CSS layout module for flexible and responsive design.
  30. How do you create a form in HTML? - Using the <form> tag with input fields.
  31. What is the purpose of the placeholder attribute? - It provides a hint in an input field.
  32. How do you comment in HTML? - Using `<!-- comment -->`.
  33. How do you comment in CSS? - Using `/* comment */`.
  34. How do you comment in JavaScript? - Using `//` for single-line and `/* */` for multi-line.
  35. What is the difference between <strong> and <b>? - <strong> gives semantic importance, while <b> is just bold styling.
  36. What is the difference between <em> and <i>? - <em> adds emphasis, while <i> is just italic styling.
  37. What is a viewport in responsive design? - It defines how a webpage scales on different devices.
